### 1.说一下常见的Linux命令? 

:::note

- `cd` 切换目录   `ls` 查看当前目录的内容    `pwd` 查看当前目录
- `mkdir` 创建文件夹    `rmdir` 删除文件夹
- `mv` 移动    `cp` 复制  `rm` 删除文件    `cat` 查看文件内容   `head` 显示文件头部内容
- `zip`、`unzip` 压缩、解压
- `ps` 查看进程    `netstat` 查看端口占用    ` kill -9` 杀掉进程
- `top` 显示系统进程占用资源情况     `nvidia-smi` 查看GPU占用情况
- `find` 查找文件   `grep` 过滤文件

:::

### 2.linux怎么解压一个文件？如果只有一个tar文件怎么解压？

:::note

Linux的解压缩命令有：tar 命令、zip命令 和 rar 命令。

压缩：

tar –cvf jpg.tar *.jpg       // 将目录里所有jpg文件打包成 jpg.tar

tar –czf jpg.tar.gz *.jpg    // 将目录里所有jpg文件打包成 jpg.tar 后，并且将其用 gzip 压缩，生成一个 gzip 压缩过的包，命名为 jpg.tar.gz 

rar a jpg.rar *.jpg          // rar格式的压缩

zip jpg.zip *.jpg            // zip格式的压缩

解压：

tar –xvf file.tar         // 解压 tar 包 

tar -xzvf file.tar.gz     // 解压 tar.gz 

tar -xjvf file.tar.bz2    // 解压 tar.bz2 

tar –xZvf file.tar.Z      // 解压 tar.Z 

unrar e file.rar          // 解压 rar 

unzip file.zip            // 解压 zip 

:::

### 4.你项目是怎么打包和部署的？运行jar包时怎么把参数传递给主函数？

:::note

打包是使用maven进行打包的，打包成jar包。之后上传到服务器，通过java -jar命令运行。

可以直接在java -jar x.jar后面直接传递参数，比如java -jar x.jar a b 会通过main()方法的args数组进行传递。java -jar x.jar --a=111 --b=2222 也可以。参数和jar包之间 参数和参数之间用空格隔开。

:::
