## Linux

### 说一下常见的Linux命令? 

:::note

- `cd` 切换目录   `ls` 查看当前目录的内容    `pwd` 查看当前目录
- `mkdir` 创建文件夹    `rmdir` 删除文件夹
- `mv` 移动    `cp` 复制  `rm` 删除文件    `cat` 查看文件内容   `head` 显示文件头部内容
- `zip`、`unzip` 压缩、解压
- `ps` 查看进程    `netstat` 查看端口占用    ` kill -9` 杀掉进程
- `top` 显示系统进程占用资源情况     `nvidia-smi` 查看GPU占用情况
- `find` 查找文件   `grep` 过滤文件

:::

### 如何查询端口占用情况？

:::note

可以使用`lsof` 和`netstat` 命令：

- `sudo lsof -i :80`
- `netstat -tunlp | grep :80`

`lsof -i` 用于查看网络套接字连接情况，清晰明了。`netstat` 是经典网络工具。

:::

### linux怎么解压一个文件？如果只有一个tar文件怎么解压？

:::note

Linux的解压缩命令有：tar 命令、zip命令 和 rar 命令。

压缩：

tar –cvf jpg.tar *.jpg       // 将目录里所有jpg文件打包成 jpg.tar

tar –czf jpg.tar.gz *.jpg    // 将目录里所有jpg文件打包成 jpg.tar 后，并且将其用 gzip 压缩，生成一个 gzip 压缩过的包，命名为 jpg.tar.gz 

rar a jpg.rar *.jpg          // rar格式的压缩

zip jpg.zip *.jpg            // zip格式的压缩

解压：

tar –xvf file.tar         // 解压 tar 包 

tar -xzvf file.tar.gz     // 解压 tar.gz 

tar -xjvf file.tar.bz2    // 解压 tar.bz2 

tar –xZvf file.tar.Z      // 解压 tar.Z 

unrar e file.rar          // 解压 rar 

unzip file.zip            // 解压 zip 

:::

## Web开发

### cookie和session的区别？

参考资料：

https://www.cnblogs.com/ityouknow/p/10856177.html

https://blog.csdn.net/weixin_45393094/article/details/104747360

:::note

- Cookie 存储在客户端（浏览器），而Session 存储在服务器端
- Session比Cookie更安全
- Cookie 只能保存 ASCII，Session 可以存任意数据类型
- Cookie 可设置为长时间保持，Session 一般失效时间较短，客户端关闭会失效
- 单个 Cookie 保存的数据不能超过 4K，Session 可存储数据的大小与服务器的内存有关

:::

### 过滤器和拦截器的区别？

参考资料：

https://www.cnblogs.com/vipstone/p/16796867.html

https://www.cnblogs.com/summerday152/p/13658788.html

:::note

- 过滤器来自于 Servlet，而拦截器来自于 Spring 框架；
- 过滤器是基于函数回调实现，拦截器是基于Java的反射机制实现；
- 过滤器对请求进入Servlet前后进行处理，拦截器对请求进入Controller前后进行处理，所以过滤器会先执行，然后才会执行拦截器，最后才会进入真正的要调用的方法；
- 过滤器依赖 Servlet 容器，只能用在 Web 项目中，拦截器即可以用于web项目也可以用于非web项目；
- 过滤器通常用来实现通用功能过滤，如敏感词过滤，拦截器主要用来实现项目中的业务判断，如登录判断、权限判断；

:::
