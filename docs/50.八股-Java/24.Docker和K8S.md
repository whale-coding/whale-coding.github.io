### 1.说一下Docker的基础命令?

:::note

- `docker pull` 拉取镜像
- `docker push`推送镜像
- `docker images`列出本地镜像
- `docker rmi` 删除镜像
- `docker run`创建并运行容器
- `docker ps`列出容器
- `docker start/stop/restart`启动/停止/重启容器
- `docker rm`删除容器
- `docker exec`在容器中执行命令
- `docker build`根据Dockerfile 构建镜像

:::
