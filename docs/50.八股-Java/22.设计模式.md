### 6.什么是工厂模式？

工厂模式（Factory Pattern）是一种**创建型设计模式**，其核心思想是**将对象的创建逻辑从使用逻辑中分离出来**——就像现实中的工厂负责生产产品一样，代码中的"工厂"专门负责实例化对象，而调用者只需要"消费"产品，无需关心制造过程。
